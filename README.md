# Career Roadmap Analyzer 🚀

An intelligent career guidance web application that analyzes user skills, calculates job readiness, identifies missing skills, and generates personalized learning roadmaps for various tech careers.

---

## 📌 Features

- Skill Gap Analysis
- Job Readiness Percentage
- Interactive Doughnut Charts
- Personalized Learning Roadmap
- Multiple Career Roles Support
- Real-Time Skill Comparison
- Responsive User Interface

---

## 💻 Supported Roles

- Frontend Developer
- Backend Developer
- Full Stack Developer
- Data Analyst
- UI/UX Designer
- AI Engineer
- Cybersecurity Analyst

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Chart.js

---

## 📊 How It Works

1. Enter your existing skills
2. Select your target role
3. Click on "Analyze Skills"
4. View:
   - Missing skills
   - Job readiness %
   - Skill comparison chart
   - Personalized roadmap
